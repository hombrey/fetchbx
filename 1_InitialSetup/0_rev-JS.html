If you're on team Windows, do ths:
  1. activate WSL1
  2. prepare shtaje environment
  3. build and run shtaje

↓ ↓ ↓

1. activate WSL

1.1. Enable Windows Features: (a) WSL and (b)VM Platform.

   - Go to "Turn Windows features on or off" (search for it in the Start menu). (see here)
   - Check the boxes next to "Windows Subsystem for Linux (WSL)" and "Virtual Machine Platform".
   - Click "OK" and wait for the process to complete.

1.2. Restart the computer

1.3. Open Command Prompt as an Administrator

   - Search for "cmd" in the Start menu.
   - Right-click on "Command Prompt" and select "Run as administrator".

1.4. Execute this command to ensure WSL1:


wsl --set-default-version 1

1.5. Run this command to install Ubuntu:


wsl --install -d Ubuntu

1.6. Provide new USERNAME and PASSWORD

1.7. You may exit the terminal after this

2. prepare shtaje environment

2.1. Launch 'Ubuntu' from the start menu

2.2. run this command(to update ubuntu): (click to copy)


sudo apt update && sudo apt upgrade -y && sudo apt install unzip -y

2.3. run this command(get and unzip baxtaje): (click to copy) 


cd ~ && curl https://hombrey.github.io/fetchbx/baxtaje.zip -o baxtaje.zip\ 
&& unzip baxtaje.zip && rm baxtaje.zip

2.4. run this command (install apps): (click to copy) 


~/baxtaje/Scripts/nixtall/noNix.sh

2.5. exit the 'Ubuntu' program

3. build and run shtaje

3.1. launch "Ubuntu" from the start menu

3.2. run this command (verify nix installation) 


nix --version

3.3. Finally, run this command to enter into vifm (the initial setup will take time): 


shtaje

Team mac, follow these steps:
  1. prepare nix+shtaje environment
  2. build and run shtaje

↓ ↓ ↓

1. prepare nix and shtaje environment

1.1. launch the terminal

1.2. run this command(get and unzip baxtaje): (click to copy) 


cd ~ && curl https://hombrey.github.io/fetchbx/baxtaje.zip -o baxtaje.zip\ 
&& unzip baxtaje.zip && rm baxtaje.zip

1.3. run this command (install nix): (click to copy) 


~/baxtaje/Scripts/nixtall/onMac.sh

1.4. exit the terminal

2. pull shtajebuilder container

2.1. re-launch the terminal app

2.2. run this command (verify nix installation) 


nix --version

2.3. Finally, run this command to enter into vifm (the initial setup will take time): 


shtaje
