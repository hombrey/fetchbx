basic navigation (←)

key function
<↑> , <k> item above
<↓> , <j> item below
<←> , <h> parent directory
<→> , <l> open
<tab> switch pane

system keybindings (←)

key function
<esc> normal mode
<space> command mode
<v> select mode
<ctrl-d> , <,q> quit
<,1> light theme
<,2> dark theme

file management (copy,move,create,delete) (←)

key normal mode
<yy> yank from current location
<p> paste at current location
<P> move to current location
<,nd> new directory
<DD> delete
<yp> make a copy on next pane
<yP> move to next pane
key v-select mode
<y> yank from current location
- -
- -
- -
<D> delete
<yp> make a copy on next pane
<yP> move to next pane

file management (rename and find) (←)

key function
</> find within working directory
<n>,<N> find next / previous
<f> fuzzy find files within directory tree
<cw> rename file
<cc> rename file(s) in editor
(also works in select mode)

additional features (←)

key function

<s> go to shell
<S> sort options
